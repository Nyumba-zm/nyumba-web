# Nyumba MVP Enhancement Guide

## Overview

This document outlines the critical enhancements required to make the
Nyumba MVP strong, clear, and aligned with our core mission: building a
trusted, AI-powered, end-to-end real estate platform for Zambia.

## Key Areas for MVP Enhancement

### 1. Trust & Verification

-   Implement KYC for all agents and sellers.
-   Add document upload + internal verification for property ownership.
-   Introduce automated duplicate-listing detection.
-   Begin work on a future title-verification pipeline.

### 2. Centralized Listing System

-   Clean UI for exploring listings with filters.
-   Mandatory structured fields (price, rooms, area, amenities, etc.).
-   Add HD image support and preparation for future 360° tours.
-   Optimize search speed via indexing.

### 3. AI Valuation Engine (Early Prototype)

-   Integrate Python microservice for automated price estimation.
-   Display "Nyumba Estimate" and comparables on listing pages.
-   Begin capturing property attributes for training future ML models.

### 4. Financing & Loan Matching

-   Create a unified loan pre-qualification form.
-   Add lender profiles and comparison UI (interest rate, tenure, fees).
-   Implement ML-based lender matching logic (rule-based for MVP).

### 5. Messaging & Secure Comms

-   Build in-app buyer ↔ seller/agent messaging.
-   Add basic anti-fraud language detection (MVP-light).
-   Ensure all messages stay on-platform.

### 6. Transaction Pipeline

-   Add an "Offer → Inspection → Payment → Close" workflow pipeline.
-   Prepare API structure for future escrow integration.
-   Add checklist UI for required documents and steps.

### 7. Neighborhood Insights (MVP-Lite)

-   Add static neighborhood metadata per area:
    -   Safety perception
    -   Basic amenities
    -   Distance to city center
-   Prepare space for future data integrations.

### 8. Service Marketplace (Light Version)

-   Add categories: inspections, insurance, contractors.
-   Allow providers to create business profiles.
-   Prepare matching logic for future automation.

### 9. Content & Knowledge Center

-   Add guides: "How to rent safely", "Mortgage steps", "Avoiding
    scams".
-   Add calculators: mortgage payment, rent vs buy.

### 10. Core Enhancements for UX Quality

-   Improve listing card design.
-   Improve transitions and load states.
-   Make mobile-first UI a top priority.
-   Ensure consistent brand colors and spacing.

## Summary of MVP Goals

The MVP should: - Completely eliminate the chaotic Facebook-style search
experience. - Prove the value of verification + AI pricing. - Provide
one platform where users can search → evaluate → chat → apply for
financing. - Establish Nyumba as the safest and most reliable property
marketplace in Zambia.
